XCUI 是基于`Vue`和 `Bootstrap` 的桌面端组件库。

XCUI的目标是：

+ 满足桌面端页面大部分组件需求。
+ 支持npm install； 支持按需加载和全部加载。
+ 保持小体积(不依赖`jquery`, `Boostrap.js`), 仅使用 `Boostrap.css` 。
+ 简洁优雅。

## 使用

```
# clone the  repo
git clone git@github.com:wmfe/xcui.git

# 安装依赖
`$ npm i`

# 开始开发
`$ npm run dev `

# 编译组件
`$ npm run build`

```
